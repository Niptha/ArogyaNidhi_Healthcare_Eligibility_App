# 🏥 ArogyaNidhi – Smart Healthcare Eligibility & Assistance App

ArogyaNidhi is an Android healthcare assistance application developed using Kotlin and Jetpack Compose.  
The app helps users check eligibility for government healthcare schemes, explore available schemes, find hospitals, and securely manage important healthcare documents.

---

# ✨ Features

## 🔐 User Authentication
- Login and Registration system
- Input validation
- Error handling for invalid fields

## 📋 Eligibility Checker
- 5-Step healthcare eligibility assessment
- Animated step-by-step UI
- Dynamic eligibility results
- Personalized scheme recommendations

## 🏥 Government Schemes
- Displays multiple Indian healthcare schemes
- Eligibility details
- Benefits information
- Required documents

## 📄 Document Locker
- Upload and manage important healthcare documents
- Local Room Database integration
- Secure document storage system

## 🎨 Modern UI
- Built using Jetpack Compose
- Professional healthcare-themed interface
- Gradient backgrounds
- Material 3 Design
- Responsive layouts

## 🧠 MVVM Architecture
- ViewModel integration
- Clean architecture structure
- Better state management
